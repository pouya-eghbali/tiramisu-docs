### Task 9: Update SEO routes for multi-locale support

The SEO helpers in `packages/kit/src/seo.ts` need locale-aware URL building. The SEO routes need to iterate all locales.

**Files:**
- Modify: `packages/kit/src/seo.ts` — update `generateLlmsTxt`, `generateLlmsFullTxt`, `generateSkillMd` to accept optional `localePrefix` in their URL options
- Modify: `playground/src/routes/sitemap.xml/+server.ts`
- Modify: `playground/src/routes/llms.txt/+server.ts`
- Modify: `playground/src/routes/llms-full.txt/+server.ts`
- Modify: `playground/src/routes/skill.md/+server.ts`

**Step 1: Update `seo.ts` — add `localePrefix` option**

Add `localePrefix?: string` to the `basePath` option pattern in `generateLlmsTxt`, `generateLlmsFullTxt`, and `generateSkillMd`. When set, URLs become `{base}/docs/{localePrefix}/{slug}` instead of `{base}/docs/{slug}`.

For `generateSitemap`, it already supports `locales` and `defaultLocale` options — just need to pass them.

**Step 2: Update playground SEO routes**

Each route should import `locales` and `defaultLocale` from `virtual:tiramisu-docs` (they're `undefined` when i18n is off, so the routes stay backward-compatible).

**sitemap.xml** — pass config locales and defaultLocale:
```typescript
import { generateSitemap } from "@tiramisu-docs/kit/seo"
import { docs, locales, defaultLocale } from "virtual:tiramisu-docs"
import config from "$lib/tiramisu.config"
import { resolveConfig } from "@tiramisu-docs/kit"

export const prerender = true

export function GET() {
  const resolved = resolveConfig(config)
  const baseUrl = resolved.url ?? "https://example.com"
  const allDocs = locales
    ? Object.entries(locales).flatMap(([code, data]) =>
        data.docs.map((d) => ({ ...d, slug: `${code}/${d.slug}` }))
      )
    : docs
  const xml = generateSitemap(allDocs, {
    baseUrl,
    locales: resolved.i18n?.locales,
    defaultLocale: resolved.i18n?.defaultLocale,
  })
  return new Response(xml, { headers: { "Content-Type": "application/xml" } })
}
```

**llms.txt** — concatenate all locales:
```typescript
import { generateLlmsTxt } from "@tiramisu-docs/kit/seo"
import { docs, locales } from "virtual:tiramisu-docs"
import config from "$lib/tiramisu.config"
import { resolveConfig } from "@tiramisu-docs/kit"

export const prerender = true

export function GET() {
  const resolved = resolveConfig(config)
  const baseUrl = resolved.url ?? "https://example.com"
  if (locales) {
    const parts = Object.entries(locales).map(([code, data]) =>
      generateLlmsTxt(data.docs, {
        title: resolved.title,
        description: resolved.description,
        baseUrl,
        basePath: `/docs/${code}`,
      })
    )
    return new Response(parts.join("\n"), {
      headers: { "Content-Type": "text/plain; charset=utf-8" },
    })
  }
  const txt = generateLlmsTxt(docs, {
    title: resolved.title,
    description: resolved.description,
    baseUrl,
  })
  return new Response(txt, { headers: { "Content-Type": "text/plain; charset=utf-8" } })
}
```

**llms-full.txt** — same pattern with `generateLlmsFullTxt` and `searchIndex`.

**skill.md** — same pattern with `generateSkillMd`.
